# home

Interactive terminal-style portfolio built with React, TypeScript, and Vite.

## Commands

| Command | Description |
|---------|-------------|
| `/about` | Who am I |
| `/skills` | Technical skills |
| `/resume` | Work history |
| `/projects` | Side projects |
| `/reset` | Clear screen |
| `/help` | Show available commands |

## Development

```bash
pnpm install
pnpm dev
```
